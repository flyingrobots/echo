# ttd-manifest

Generated TTD protocol manifest and IR for Echo's local consumption of the
canonical `warp-ttd` protocol.

This is a data-only crate used to stage and provide access to the TTD
protocol's Intermediate Representation (IR) and metadata generated through the
Wesley sync path.

The contents of this directory are managed by `cargo xtask wesley sync`.
Do not treat this crate as an independent protocol source of truth.
