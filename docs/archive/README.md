# Archive

Documents that are **no longer the active source of truth** for any ongoing
work. This includes superseded specs, completed plans, abandoned proposals, and
design notes whose insights have been absorbed into code or canonical docs.

**Rule of thumb:** if someone would read a doc and ask "should I be doing
something about this?" — it does not belong here. If the answer is "this
already happened" or "we went a different direction" — archive it.
