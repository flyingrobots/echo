# Enforce design vocabulary like warp-ttd

warp-ttd has a tested glossary that enforces "worldline not timeline,"
"receipt not log entry," etc. Echo uses these terms informally but
doesn't enforce them. A glossary test would catch terminology drift
in docs and code comments.
