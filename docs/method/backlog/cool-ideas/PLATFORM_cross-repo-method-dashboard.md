# Cross-repo METHOD dashboard

Run `method status --json` across multiple repos (echo, warp-ttd,
bijou, method, git-warp) and aggregate into a single view. See
legend load and active cycles across the entire project constellation.

Could be a CLI tool, a git-mind query, or a web page.
