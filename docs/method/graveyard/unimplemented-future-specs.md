# Unimplemented future specs

**Rejected.** 17 spec files were deleted during the 2026-04-03 docs
audit. They described features that do not exist:

- ECS scheduler, storage, and world API
- Branch tree, time knots, timecube, temporal bridge
- Entropy and paradox handling
- Warp confluence and merge semantics
- Multi-writer concurrency (MWMR)
- Networking, plugin system, capability/security model
- Serialization protocol

These were aspirational documents with no implementation, no tests,
and no witnesses. They created false impressions of completeness.

If any of these features are built in the future, they should start
as backlog items with a proper cycle, not as pre-written specs.
