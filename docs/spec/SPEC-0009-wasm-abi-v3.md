# SPEC-0009: WASM ABI Contract v3 (Historical)

> **Status:** Superseded | **Historical ABI Version:** 3

ABI v3 is not a supported compatibility target and this file is not normative.
The [canonical WASM ABI specification](SPEC-0009-wasm-abi.md) defines the
current epoch and version.

The former v3 export table, wire contract, migration notes, and test checklist
remain available in Git history. They must not be used to infer current exports,
current intent envelopes, or current observation authority.
